My little piano

      How to write a melody:

      1. Use the following note names: do-1, do-diez-1, re-1, re-diez-1, mi-1, fa-1, fa-diez-1, sol-1, sol-diez-1, la-1, la-diez-1, si-1, do-2, do-diez-2, re-2, re-diez-2, mi-2;

      2. After the note name, enter the duration in milliseconds in square brackets (for a fast tempo, use 480 for a quarter like in the example mi-1[480]);

      3. To create an interval or chord, write the necessary notes in parentheses. Specify the duration in milliseconds in square brackets without a space after the closing parenthesis (for example, (do-1 mi-1 sol-1)[960] - chord "do-mi-sol" will sound 960ms);

      4. The duration of pauses between notes should be specified also in milliseconds but without parentheses (in the example mi-1[480] 360 fa-diez-1[120] sol-1[480] 360 there are 2 pauses of 360ms duration).
